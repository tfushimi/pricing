# Executables

| Executable | Description |
|---|---|
| `barrier_enhanced_note` | Prices a Barrier Enhanced Note across a range of spots using Black-Scholes with an SVI vol surface |
| `json_example` | Loads a payoff definition from `payoff.json` and prices it using Black-Scholes |

See [gatheral/README.md](gatheral/README.md) for executables reproducing figures from Gatheral (2006) *The Volatility Surface*.
